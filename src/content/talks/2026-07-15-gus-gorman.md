Gus covered how the Omaha metro's ORION P25 trunked radio network encrypts voice traffic while leaving rich metadata openly accessible. Using software-defined radio tools like SDRTrunk, OP25, and Trunk-Recorder, control-channel data - talkgroup IDs, radio unit IDs, call timestamps, and durations - reveals activity patterns without decoding a single voice packet. The talk also explored RF proximity-detection techniques: TPMS tire-pressure sensors on patrol vehicles, helicopter ADS-B transponders, drone OpenDroneID broadcasts, Axon body-camera Bluetooth beacons, and WiFi wardriving, all of which can be correlated with talkgroup activity to map an operational picture in near real-time.
